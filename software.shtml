Bi 188 Spring 2018 - Recommended software

This list of software will be constantly updated. Platform-specific software will be noted as such.

Software to download

Putty - Windows terminal for accessing Unix/Linux systems (Use built-in terminal for Mac/Linux)

Filezilla - Cross-platform FTP, for uploading/downloading files from server securely

Treeview - Visualize and compare multiple datasets, gene-by-gene, showing pre-calculated clusters

Caltech IMSS software: Where you get your Matlab. Optional.

Spyder IDE for Python - While many find a text editor to be totally adequate for coding in Python, as a coding amateur I find an IDE essential (along with a browser window open at all times).

Seqmonk - Handy tool for visualizing and analyzing post-mapping data

Tools on the web - Essentials

UCSC Genome Browser - Where you visualize and zoom into your genome-wide data at various stages

ENCODE Project Main Page - Where you download data from genome-wide experiments performed in the ENCODE project

Gencode - Where you download the latest in RNA transcriptions

Galaxy Project on the Web - Free site to process your sequencing data

Custom tsv "translate" script for translating .tsv into usable gene names, via a matching Gencode annotation file
