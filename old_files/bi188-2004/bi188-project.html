Sequence Retrieval, Analysis, and Comparison

Individual proteins/mRNA can be retrieved from NCBI's Entrez Web site.

If you are looking for cross-species matches (i.e. orthologues) to a particular protein/mRNA, you can also use NCBI's BLAST server.

The most useful way to obtain genomic sequence, however, is probably not NCBI; ENSEMBL allows you to search many of the available genomes for the genomic region corresponding to a protein. Note that chicken and chimp are available as "pre-releases".

Once you have identified the genomic region(s) you want to analyze, you should use RepeatMasker to get rid of repetitive and low-complexity sequence.

You can then do a comparative analysis with FamilyRelations. Note that there are some other programs out there that also do comparative analysis -- most notably Vista and PipMaker.

Please note: if you're using FamilyRelations, please use the public Cartwheel server, Woodward; choose the Bi188 lab area on that server. You will also need to download a new version of FRII.
